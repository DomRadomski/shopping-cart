# Shopping Cart

A simple React shopping cart application built with React Router, dynamic product routing, basket state management, and responsive UI components.

## Live Demo

View the full deployed project [here](https://shopping-cart-jet-six.vercel.app)

## Features

- Product listing
- Product modal routing
- Basket management
- Quantity selection
- Responsive design

## Tech Stack

- React
- React Router
- Vite, Vercel
- CSS Modules
